# Code Reading

A few idea on how to read the code of a project: module/library/crate


* Look at the CI to see how to run the tests
* Look at some tests to see how to use the code
* Rust: In Cargo.toml find the executables, if there are any.
* Rust: look at the `main.rs` file, if it exists
* Look for examples in the documentation or in a folder called example and follow those functions.
* Write a small example program (based on the documentation) and follow that.

* Perl: Makefile.PL
